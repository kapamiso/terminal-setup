# Dotfiles

Personal terminal setup for macOS, Linux, and Windows (WSL).

## Quick Setup

```bash
git clone git@github.com:kapamiso/terminal-setup.git ~/terminal-setup
cd ~/terminal-setup
chmod +x setup.sh
./setup.sh
```

## What's Included

- **zsh** config with Oh My Zsh and Powerlevel10k
- **p10k** theme config
- **tmux** config
- **git** config

## Windows

Use WSL (Windows Subsystem for Linux), then run the setup script inside WSL.
